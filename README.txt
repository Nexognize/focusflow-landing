FocusFlow Showcase Site

Files
- index.html
- styles.css
- script.js

Run locally (any one option):
1) Open index.html directly in a browser
2) Serve with a static server from this folder, for example:
   python3 -m http.server 8080
   Then open: http://localhost:8080

Deployment
- Upload all files in this folder to any static host:
  Netlify, Vercel (static), Cloudflare Pages, GitHub Pages, S3 static hosting, or your own VPS.
- Ensure index.html is at the root of the deployed site.

Feedback form note
- Current form is demo-only and shows an on-page confirmation.
- To collect real responses, connect the form to Formspree, Tally, Google Forms, or your backend endpoint.
